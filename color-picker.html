Frequently Asked Questions

What color formats are supported?

DevUtilities Color Picker supports 7 professional color formats: HEX (#RRGGBB and #RRGGBBAA with alpha), RGB (0-255 range), RGBA (with alpha 0-1), HSL (hue, saturation, lightness), HSLA (with alpha), HSB/HSV (hue, saturation, brightness), and CMYK (for print design).

How does the color history work?

The Color Picker automatically saves up to 22 of your most recently selected colors. Each color is displayed as a small visual swatch that you can click to quickly reload that color. The history persists between app sessions.

Can I edit color values directly?

Yes! You can type directly into any color format field. The Color Picker uses regex-based validation to parse your input and instantly updates all other format fields to reflect your changes. This is perfect for quickly adjusting specific color values.

What's the difference between HSL and HSB?

HSL (Hue, Saturation, Lightness) and HSB/HSV (Hue, Saturation, Brightness/Value) are two different color space representations. HSL uses lightness (0% = black, 50% = pure color, 100% = white), while HSB uses brightness (0% = black, 100% = pure color). Both are useful for different color manipulation tasks.

Is the CMYK conversion accurate for print?

The CMYK conversion uses standard color space algorithms to convert from RGB. While suitable for most design work, professional print production may require ICC color profile conversions for exact color matching on specific printers and paper types.
